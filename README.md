# 数据合规智能审查系统

智能合规审查工具，支持文档审查、代码审查与数据脱敏处理，内置本地法规知识库。

## 前端修改记录（对比原版）

### 首页 (`index.html`) — 全面视觉重构

原版为蓝紫色主题（Indigo `#4f46e5`），使用 Inter / Noto Sans SC 字体；新版改为杂志风格暖色调，整体设计语言如下：

#### 色彩体系
| 令牌 | 原版 | 新版 |
|------|------|------|
| 品牌色 | `#4f46e5`（Indigo） | `#006d6a`（Teal） |
| 背景 | `#f8fafc`（冷灰） | `#f4f1ea`（暖米色）+ 网点纹理 |
| 卡片背景 | `#ffffff` | `rgba(255,253,248,0.92)` |
| 正文 | `#0f172a` | `#161a1d` |
| 边框 | `#e2e8f0` | `#d9d1c3` |
| 阴影 | 轻微蓝紫投影 | 柔和大地色投影 |

#### 字体体系
- 移除 Google Fonts（Plus Jakarta Sans / Inter / Noto Sans SC），改为系统原生字体栈
- 新增 `--font-ui`、`--font-display`、`--font-reading`、`--font-mono` 四个字体变量
- 字重从 `800` 调整为 `760`/`720`，字号从 `24px`→`22px`（导航栏 Logo），正文 `16px`

#### 布局结构
- **新增 raven-layout 侧边栏模式**：页面支持左侧固定导航栏（`width: 264px`），包含品牌标识、工作区切换（脱敏/审查）、历史记录、返回首页按钮
- **新增 home-mode 全屏展示模式**：页面可切换为大幅品牌展示界面，包含大字标题（`clamp(52px, 6vw, 88px)`）、英文描述、快速操作按钮、装饰性视觉组件
- 上传面板从单列布局改为双栏（左侧产品卡片选择 + 右侧上传表单）
- 工作区切换支持「数据脱敏」「合规审查」两个独立产品线

#### 新增组件
- **历史记录面板**：侧边栏内嵌最近任务列表，支持搜索、隐藏/显示、单条删除
- **进度弹窗**：提交任务后显示模态进度条，带指数平滑动画和检查点列表
- **错误弹窗**：提交失败时显示错误信息
- **主题切换**：暗色模式支持，浮动按钮（`theme-float`）在 home-mode 下固定在右上角
- **返回首页按钮**：侧边栏底部固定按钮，样式为黑色实底白字

#### 暗色模式
- 完整 `body.dark` 变量覆盖，所有组件均有对应的暗色样式
- `body.dark.raven-layout` 双重限定确保侧边栏模式下暗色正确

#### 响应式
- 新增 `max-width: 860px` 断点：侧边栏折叠为普通顶部导航，历史记录隐藏
- 新增 `max-width: 620px` 断点：网格布局转为单列，状态指示器隐藏

---

### 审查结果页 (`result.html`) — 侧边栏重构 + 色彩统一

原版为蓝紫色主题，顶部通栏导航（sticky header）+ 左侧粘性侧边栏。新版改为与首页一致的固定侧边栏设计。

#### 侧边栏重设计
| 特性 | 原版 | 新版 |
|------|------|------|
| 定位 | `position: sticky; top: 100px` | `position: fixed; top: 52px; bottom: 0` |
| 宽度 | 由 grid 决定（`220px`） | 固定 `264px`，与首页侧边栏一致 |
| 布局 | 普通流式 | Flexbox 列布局，主内容区 `flex:1`，底部按钮固定 |
| 层级 | 无 z-index | `z-index: 51`（高于 header 的 `50`） |

#### 侧边栏内容（自上而下）
1. **文档名称**（`.sidebar-doc`）— 显示当前审查文档名，`margin-bottom: 22px`
2. **导航标题** — "导航"标签，与首页工作区标签风格一致
3. **导航列表** — 概览、风险详情（可折叠子项）、风险聚类、整改任务、证据清单、专项审查包
4. **导出区域** — 分隔线 + "导出"标签，下载链接带图标和边框按钮样式
5. **返回上一页按钮** — 黑色实底白字，固定在侧边栏最底部

#### 新增功能
- **风险详情折叠**：点击"风险详情"展开/收起子导航项，带 SVG chevron 旋转动画，CSS transition 控制展开（`max-height` + `opacity`）
- **返回上一页按钮**：固定在侧边栏底部，跳转至 `/?workspace=review`
- **IntersectionObserver 导航高亮**：滚动页面时自动高亮当前章节对应的导航项（含风险子项）
- **移动端抽屉菜单**：`max-width: 760px` 时侧边栏变为 slide-out drawer，带半透明 backdrop 和浮动打开按钮
- **暗色模式完整覆盖**：`body.dark` 下所有组件（风险卡片、分析框、法规卡片、下载链接等）均有对应暗色样式

#### 色彩统一（与首页对齐）
- `--bg`: `#ffffff`（原 `#ffffff`，维持）、`--border`: `#d4d4d0`（原 `#e2e8f0`）
- `--accent`: `#111111`（原 `#4f46e5`）、`--accent-light`: `#f0f0ed`（原 `#e0e7ff`）
- `--text-primary`: `#111111`（原 `#0f172a`）
- 页面背景从 `var(--bg-subtle)` 改为 `#f5f5f3`
- 正文行高从 `1.6` 调整为 `1.65`，字号 `16px`

#### 下载链接样式
- 原版为纯文字链接，新版改为带边框按钮样式（`border: 1px solid var(--border)`）
- 默认使用 `background: var(--bg)` + `color: var(--text-secondary)`
- 去除默认 primary 黑色样式（`完整报告 JSON` 不再默认黑色）

#### Logo 统一
- 与首页对齐：字号 `26px`、字重 `760`、无额外 `line-height`
- `.header-inner` padding 与首页侧边栏一致（`14px 16px`）
- 主题切换按钮布局统一为 `justify-content: space-between; gap: 8px`

## 项目结构

```
data-compliance-review/
├── data-compliance-web/        # Web 应用
│   ├── app.py                  # Flask 后端
│   ├── templates/
│   │   ├── index.html          # 首页（上传/脱敏入口）
│   │   └── result.html         # 审查结果页
│   └── static/
├── scripts/                    # 工具脚本
├── test_output/                # 测试数据
├── skills/                     # Claude Code skill
├── extension/                  # 浏览器扩展

```

## 快速开始

```bash
cd data-compliance-web
python app.py
```

访问 http://127.0.0.1:5000 即可使用。
